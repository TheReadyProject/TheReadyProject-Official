O que é Java e suas aplicações:

Java é uma linguagem de programação orientada a objetos, de propósito geral, criada por James Gosling e lançada pela Sun Microsystems em 1995. É amplamente utilizada para desenvolvimento de aplicações web, móveis, de desktop e corporativas.

Instalando Java (Windows, macOS, Linux):

Windows:

  1. Acesse o site oficial do Java: oracle.com/java.
  2. Baixe o instalador do JDK (Java Development Kit).
  3. Execute o instalador e siga as instruções na tela.

macOS:

  1. Acesse o site oficial do Java.
  2. Baixe o instalador apropriado para macOS.
  3. Execute o instalador e siga as instruções na tela.

Linux:

sudo apt update
sudo apt install openjdk-11-jdk

Configuração do ambiente de desenvolvimento (IDEs recomendadas: Eclipse, IntelliJ IDEA):

Eclipse:

  1. Baixe e instale o Eclipse: eclipse.org/downloads.
  2. Crie um novo projeto Java.

IntelliJ IDEA:

  1. Baixe e instale o IntelliJ IDEA: jetbrains.com/idea.
  2. Configure um novo projeto Java.

Aula 2: Variáveis e Tipos de Dados

Objetivo: Compreender variáveis e tipos de dados em Java.

Conteúdo:

Variáveis: definição e convenções de nomenclatura:

Variáveis são espaços na memória do computador onde podemos armazenar dados. Em Java, é necessário declarar o tipo de uma variável.

String nome = "Alice";
int idade = 30;

Convenções de nomenclatura:

  • Use nomes descritivos.
  • Utilize camelCase para nomes de variáveis e métodos.

Tipos de dados básicos: inteiros, floats, strings, booleanos:

int num = 10;  // Inteiro
double pi = 3.14;  // Float
String saudacao = "Olá, Mundo!";  // String
boolean isStudent = true;  // Booleano

Operações básicas com variáveis:

int soma = 10 + 5;  // Aritmética
boolean maior = 10 > 5;  // Comparação

Função `instanceof` e conversão de tipos:

System.out.println(nome instanceof String);  // Saída: true
String numStr = "123";
int numInt = Integer.parseInt(numStr);

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Digite um número: ");
        double numero = scanner.nextDouble();
        if (numero > 0) {
            System.out.println("O número é positivo.");
        } else if (numero < 0) {
            System.out.println("O número é negativo.");
        } else {
            System.out.println("O número é zero.");
        }
        scanner.close();
    }
}

Atividade:

Criação de um programa que gerencia uma lista de tarefas, permitindo adicionar, remover e visualizar itens.

import java.util.ArrayList;
import java.util.List;
import java.util.Scanner;

public class Tarefas {
    public static void main(String[] args) {
        List tarefas = new ArrayList<>();
        Scanner scanner = new Scanner(System.in);
        
        while (true) {
            System.out.println("\nLista de Tarefas:");
            for (String tarefa : tarefas) {
                System.out.println("- " + tarefa);
            }
            
            System.out.println("\nOpções:");
            System.out.println("1. Adicionar tarefa");
            System.out.println("2. Remover tarefa");
            System.out.println("3. Sair");

            System.out.print("Escolha uma opção: ");
            String opcao = scanner.nextLine();

            if (opcao.equals("1")) {
                System.out.print("Digite a tarefa: ");
                String tarefa = scanner.nextLine();
                tarefas.add(tarefa);
            } else if (opcao.equals("2")) {
                System.out.print("Digite a tarefa a remover: ");
                String tarefa = scanner.nextLine();
                if (tarefas.contains(tarefa)) {
                    tarefas.remove(tarefa);
                } else {
                    System.out.println("Tarefa não encontrada.");
                }
            } else if (opcao.equals("3")) {
                break;
            } else {
                System.out.println("Opção inválida.");
            }
        }
        scanner.close();
    }
}

Atividade:

Criação de uma calculadora simples com funções para adição, subtração, multiplicação e divisão.

import java.util.Scanner;

public class Calculadora {
    public static int adicionar(int a, int b) {
        return a + b;
    }

    public static int subtrair(int a, int b) {
        return a - b;
    }

    public static int multiplicar(int a, int b) {
        return a * b;
    }

    public static double dividir(int a, int b) {
        if (b == 0) {
            throw new ArithmeticException("Divisão por zero não permitida");
        }
        return (double) a / b;
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Digite o primeiro número: ");
        int num1 = scanner.nextInt();

        System.out.print("Digite o segundo número: ");
        int num2 = scanner.nextInt();

        System.out.println("Escolha a operação:");
        System.out.println("1. Adição");
        System.out.println("2. Subtração");
        System.out.println("3. Multiplicação");
        System.out.println("4. Divisão");
        int escolha = scanner.nextInt();

        switch (escolha) {
            case 1:
                System.out.println("Resultado: " + adicionar(num1, num2));
                break;
            case 2:
                System.out.println("Resultado: " + subtrair(num1, num2));
                break;
            case 3:
                System.out.println("Resultado: " + multiplicar(num1, num2));
                break;
            case 4:
                try {
                    System.out.println("Resultado: " + dividir(num1, num2));
                } catch (ArithmeticException e) {
                    System.out.println(e.getMessage());
                }
                break;
            default:
                System.out.println("Operação inválida.");
                break;
        }
        scanner.close();
    }
}

Encapsulamento:

public class Pessoa {
    private String nome;
    private int idade;

    public Pessoa(String nome, int idade) {
        this.nome = nome;
        this.idade = idade;
    }

    public String getNome() {
        return nome;
    }

    public void setNome(String nome) {
        this.nome = nome;
    }

    public int getIdade() {
        return idade;
    }

    public void setIdade(int idade) {
        this.idade = idade;
    }

    public void apresentar() {
        System.out.println("Olá, meu nome é " + nome + " e tenho " + idade + " anos.");
    }
}

public class Main {
    public static void main(String[] args) {
        Pessoa pessoa = new Pessoa("Alice", 30);
        pessoa.apresentar();
        pessoa.setNome("Bob");
        pessoa.setIdade(25);
        pessoa.apresentar();
    }
}

Atividade:

Criação de uma classe `Carro` com atributos (marca, modelo, ano) e métodos (exibir informações, acelerar, frear).

public class Carro {
    private String marca;
    private String modelo;
    private int ano;

    public Carro(String marca, String modelo, int ano) {
        this.marca = marca;
        this.modelo = modelo;
        this.ano = ano;
    }

    public void exibirInformacoes() {
        System.out.println("Marca: " + marca + ", Modelo: " + modelo + ", Ano: " + ano);
    }

    public void acelerar() {
        System.out.println("O carro está acelerando.");
    }

    public void frear() {
        System.out.println("O carro está freando.");
    }

    public static void main(String[] args) {
        Carro carro = new Carro("Toyota", "Corolla", 2020);
        carro.exibirInformacoes();
        carro.acelerar();
        carro.frear();
    }
}
