O que é Python e suas aplicações:

Python é uma linguagem de programação de alto nível, interpretada e de propósito geral. Criada por Guido van Rossum e lançada em 1991, Python é conhecida por sua sintaxe simples e legível, o que a torna ideal para iniciantes. Python é amplamente utilizada em desenvolvimento web, automação de scripts, análise de dados, inteligência artificial, desenvolvimento de jogos e muito mais.

Instalando Python (Windows, macOS, Linux):

Windows:

  1. Acesse o site oficial do Python: python.org.
  2. Baixe o instalador apropriado para Windows.
  3. Execute o instalador e certifique-se de marcar a opção "Add Python to PATH".
  4. Conclua a instalação.

macOS:

  1. Acesse o site oficial do Python.
  2. Baixe o instalador apropriado para macOS.
  3. Execute o instalador e siga as instruções na tela.

Linux:

sudo apt update
sudo apt install python3

Configuração do ambiente de desenvolvimento (IDEs recomendadas: VS Code, PyCharm):

VS Code:

  1. Baixe e instale o Visual Studio Code: code.visualstudio.com.
  2. Instale a extensão Python: Vá para a aba Extensões (Ctrl+Shift+X) e procure por "Python".

PyCharm:

  1. Baixe e instale o PyCharm: jetbrains.com/pycharm.
  2. Configure um novo projeto Python.

Aula 2: Variáveis e Tipos de Dados

Objetivo: Compreender variáveis e tipos de dados em Python.

Conteúdo:

Variáveis: definição e convenções de nomenclatura:

Variáveis são espaços na memória do computador onde podemos armazenar dados. Em Python, não é necessário declarar o tipo de uma variável, pois a linguagem é dinamicamente tipada.

nome = "Alice"
idade = 30

Convenções de nomenclatura:

  • Use nomes descritivos.
  • Utilize letras minúsculas e underscores para separar palavras (snake_case).

Tipos de dados básicos: inteiros, floats, strings, booleanos:

num = 10  # Inteiro
pi = 3.14  # Float
saudacao = "Olá, Mundo!"  # String
is_student = True  # Booleano

Operações básicas com variáveis:

soma = 10 + 5  # Aritmética
maior = 10 > 5  # Comparação

Função `type()` e conversão de tipos:

print(type(nome))  # Saída: 
num_str = "123"
num_int = int(num_str)

Aula 3: Estruturas de Controle (if, else, elif)

Objetivo: Aprender a utilizar estruturas de controle de fluxo.

Conteúdo:

Estruturas condicionais: `if`, `else`, `elif`:

if idade >= 18:
    print("Você é maior de idade.")
elif idade > 12:
    print("Você é adolescente.")
else:
    print("Você é criança.")

Exemplos práticos:

numero = 7
if numero % 2 == 0:
    print("O número é par.")
else:
    print("O número é ímpar.")

Operadores de comparação e lógicos:

idade = 20
if idade >= 18 and idade < 65:
    print("Você é um adulto.")

Atividade:

Criação de um script simples que solicita ao usuário um número e verifica se ele é positivo, negativo ou zero.

numero = float(input("Digite um número: "))
if numero > 0:
    print("O número é positivo.")
elif numero < 0:
    print("O número é negativo.")
else:
    print("O número é zero.")

Módulo 2: Estruturas de Dados

Aula 1: Listas, Tuplas e Dicionários

Objetivo: Compreender e utilizar listas, tuplas e dicionários.

Conteúdo:

Listas:

frutas = ['maçã', 'banana', 'laranja']
print(frutas[0])  # Saída: maçã

Operações com listas:

frutas.append('morango')  # Adiciona um elemento ao final da lista
frutas.remove('banana')  # Remove um elemento da lista

Tuplas:

ponto = (10, 20)
x, y = ponto  # Desempacotamento de tupla

Características:

  • Imutáveis
  • Usadas para armazenar dados heterogêneos

Dicionários:

dados_aluno = {'nome': 'João', 'idade': 25, 'curso': 'Engenharia'}
print(dados_aluno['idade'])  # Saída: 25

Operações com dicionários:

dados_aluno['idade'] = 26  # Atualiza o valor de uma chave
dados_aluno.pop('curso')  # Remove um par chave-valor

Atividade:

Criação de um programa que gerencia uma lista de tarefas, permitindo adicionar, remover e visualizar itens.

tarefas = []

while True:
    print("\nLista de Tarefas:")
    for tarefa in tarefas:
        print(f"- {tarefa}")

    print("\nOpções:")
    print("1. Adicionar tarefa")
    print("2. Remover tarefa")
    print("3. Sair")

    opcao = input("Escolha uma opção: ")

    if opcao == "1":
        tarefa = input("Digite a tarefa: ")
        tarefas.append(tarefa)
    elif opcao == "2":
        tarefa = input("Digite a tarefa a remover: ")
        if tarefa in tarefas:
            tarefas.remove(tarefa)
        else:
            print("Tarefa não encontrada.")
    elif opcao == "3":
        break
    else:
        print("Opção inválida.")

Módulo 3: Funções e Módulos

Aula 1: Definição de Funções

Objetivo: Aprender a criar e utilizar funções em Python.

Conteúdo:

Definição e chamada de funções:

def saudacao(nome):
    print(f"Olá, {nome}!")

saudacao("Alice")  # Saída: Olá, Alice!

Argumentos e parâmetros:

def soma(a, b):
    return a + b

resultado = soma(3, 5)
print(resultado)  # Saída: 8

Parâmetros padrão e argumentos nomeados:

def saudacao(nome="Visitante"):
    print(f"Olá, {nome}!")

saudacao()  # Saída: Olá, Visitante!
saudacao("Alice")  # Saída: Olá, Alice!

Aula 2: Módulos e Pacotes

Objetivo: Aprender a organizar código em módulos e pacotes.

Importação de módulos:

import math
print(math.sqrt(16))  # Saída: 4.0

Criação e importação de módulos personalizados:

# No arquivo meu_modulo.py
def saudacao(nome):
    print(f"Olá, {nome}!")

# No arquivo principal
import meu_modulo

meu_modulo.saudacao("Alice")  # Saída: Olá, Alice!

Pacotes:

# Estrutura de diretórios
# meu_pacote/
# ├── __init__.py
# ├── modulo1.py
# └── modulo2.py

# No arquivo __init__.py
from .modulo1 import funcao1
from .modulo2 import funcao2

Atividade:

Criação de um módulo personalizado que contém funções para operações matemáticas básicas (soma, subtração, multiplicação, divisão) e um programa que utiliza essas funções.

# No arquivo operacoes.py
def soma(a, b):
    return a + b

def subtracao(a, b):
    return a - b

def multiplicacao(a, b):
    return a * b

def divisao(a, b):
    if b != 0:
        return a / b
    else:
        return "Divisão por zero não é permitida"

# No arquivo principal
import operacoes

print(operacoes.soma(3, 5))         # Saída: 8
print(operacoes.subtracao(10, 2))   # Saída: 8
print(operacoes.multiplicacao(4, 6))# Saída: 24
print(operacoes.divisao(20, 4))     # Saída: 5.0
