Curso de Programação em C#

1. C#

Módulo 1: Fundamentos de C#

Aula 1: Introdução e Configuração do Ambiente

Objetivo: Introduzir o C# e configurar o ambiente de desenvolvimento.

Conteúdo:

O que é C# e suas aplicações:

C# é uma linguagem de programação moderna, orientada a objetos, desenvolvida pela Microsoft. Ela é amplamente utilizada para desenvolvimento de aplicações Windows, jogos com Unity, e aplicativos web com ASP.NET.

Instalando Visual Studio para desenvolvimento em C#:

  1. Acesse o site oficial do Visual Studio: visualstudio.microsoft.com.
  2. Baixe e instale a versão apropriada para o seu sistema operacional.
  3. Durante a instalação, selecione a carga de trabalho "Desenvolvimento para desktop com .NET".

Configuração do ambiente de desenvolvimento:

Após a instalação, abra o Visual Studio e crie um novo projeto C#.

Primeiro programa: "Hello, World!":

using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello, World!");
    }
}

Aula 2: Variáveis e Tipos de Dados

Objetivo: Compreender variáveis e tipos de dados em C#.

Conteúdo:

Variáveis: definição e convenções de nomenclatura:

string nome = "Alice";
int idade = 30;

Convenções de nomenclatura:

  • Use nomes descritivos.
  • Utilize camelCase para nomes de variáveis e PascalCase para nomes de classes e métodos.

Tipos de dados básicos: inteiros, floats, strings, booleanos:

int num = 10;  // Inteiro
double pi = 3.14;  // Float
string saudacao = "Olá, Mundo!";  // String
bool isStudent = true;  // Booleano

Operações básicas com variáveis:

int soma = 10 + 5;  // Aritmética
bool maior = 10 > 5;  // Comparação

Função `GetType` e conversão de tipos:

Console.WriteLine(nome.GetType());  // Saída: System.String
string numStr = "123";
int numInt = int.Parse(numStr);

Aula 3: Estruturas de Controle (if, else, else if)

Objetivo: Aprender a utilizar estruturas de controle de fluxo.

Conteúdo:

Estruturas condicionais: `if`, `else`, `else if`:

if (idade >= 18)
{
    Console.WriteLine("Você é maior de idade.");
}
else if (idade > 12)
{
    Console.WriteLine("Você é adolescente.");
}
else
{
    Console.WriteLine("Você é criança.");
}

Exemplos práticos:

int numero = 7;
if (numero % 2 == 0)
{
    Console.WriteLine("O número é par.");
}
else
{
    Console.WriteLine("O número é ímpar.");
}

Operadores de comparação e lógicos:

int idade = 20;
if (idade >= 18 && idade < 65)
{
    Console.WriteLine("Você é um adulto.");
}

Atividade:

Criação de um script simples que solicita ao usuário um número e verifica se ele é positivo, negativo ou zero.

Console.Write("Digite um número: ");
double numero = double.Parse(Console.ReadLine());
if (numero > 0)
{
    Console.WriteLine("O número é positivo.");
}
else if (numero < 0)
{
    Console.WriteLine("O número é negativo.");
}
else
{
    Console.WriteLine("O número é zero.");
}

Módulo 2: Estruturas de Dados

Aula 1: Arrays, Listas e Dicionários

Objetivo: Compreender e utilizar arrays, listas e dicionários.

Conteúdo:

Arrays:

string[] frutas = { "maçã", "banana", "laranja" };
Console.WriteLine(frutas[0]);  // Saída: maçã

Operações com arrays:

frutas[2] = "morango";  // Atualiza o valor no índice 2

Listas:

List frutas = new List { "maçã", "banana", "laranja" };
frutas.Add("morango");  // Adiciona um elemento ao final da lista
frutas.Remove("banana");  // Remove um elemento da lista

Dicionários:

Dictionary dadosAluno = new Dictionary
{
    { "nome", "João" },
    { "idade", 25 },
    { "curso", "Engenharia" }
};
Console.WriteLine(dadosAluno["idade"]);  // Saída: 25

Operações com dicionários:

dadosAluno["idade"] = 26;  // Atualiza o valor de uma chave
dadosAluno.Remove("curso");  // Remove um par chave-valor

Atividade:

Criação de um programa que gerencia uma lista de tarefas, permitindo adicionar, remover e visualizar itens.

List tarefas = new List();

while (true)
{
    Console.WriteLine("\nLista de Tarefas:");
    foreach (string tarefa in tarefas)
    {
        Console.WriteLine($"- {tarefa}");
    }

    Console.WriteLine("\nOpções:");
    Console.WriteLine("1. Adicionar tarefa");
    Console.WriteLine("2. Remover tarefa");
    Console.WriteLine("3. Sair");

    string opcao = Console.ReadLine();

    if (opcao == "1")
    {
        Console.Write("Digite a tarefa: ");
        string tarefa = Console.ReadLine();
        tarefas.Add(tarefa);
    }
    else if (opcao == "2")
    {
        Console.Write("Digite a tarefa a remover: ");
        string tarefa = Console.ReadLine();
        if (tarefas.Contains(tarefa))
        {
            tarefas.Remove(tarefa);
        }
        else
        {
            Console.WriteLine("Tarefa não encontrada.");
        }
    }
    else if (opcao == "3")
    {
        break;
    }
    else
    {
        Console.WriteLine("Opção inválida.");
    }
}

Módulo 3: Funções e Módulos

Aula 1: Definição de Funções

Objetivo: Aprender a criar e utilizar funções em C#.

Conteúdo:

Definição e chamada de funções:

void Saudacao(string nome)
{
    Console.WriteLine($"Olá, {nome}!");

Saudacao("Alice");  // Saída: Olá, Alice!

Argumentos e parâmetros:

int Soma(int a, int b)
{
    return a + b;
}

int resultado = Soma(3, 5);

Parâmetros padrão e argumentos nomeados:

void Saudacao(string nome = "Visitante")
{
    Console.WriteLine($"Olá, {nome}!");
}

Saudacao();  // Saída: Olá, Visitante!
Saudacao("Alice");  // Saída: Olá, Alice!

Aula 2: Módulos e Pacotes

Objetivo: Aprender a organizar código em módulos e pacotes.

Importação de namespaces:

using System;
using System.Collections.Generic;

namespace ExemploNamespace
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Exemplo de namespace e importações.");
        }
    }
}

Criação e uso de módulos:

// Arquivo: Calculos.cs
namespace Utilitarios
{
    public class Calculos
    {
        public static int Soma(int a, int b)
        {
            return a + b;
        }
    }
}

// Arquivo: Program.cs
using System;
using Utilitarios;

class Program
{
    static void Main(string[] args)
    {
        int resultado = Calculos.Soma(3, 5);
        Console.WriteLine($"Resultado da soma: {resultado}");
    }
}

Atividade:

Criação de um módulo de utilitários com funções para cálculos matemáticos e sua utilização em um programa principal.

// Arquivo: Utilitarios.cs
namespace Utilitarios
{
    public class Matematica
    {
        public static int Multiplicacao(int a, int b)
        {
            return a * b;
        }

        public static int Divisao(int a, int b)
        {
            if (b == 0)
                throw new DivideByZeroException("O divisor não pode ser zero.");
            return a / b;
        }
    }
}

// Arquivo: Program.cs
using System;
using Utilitarios;

class Program
{
    static void Main(string[] args)
    {
        try
        {
            int resultadoMult = Matematica.Multiplicacao(4, 5);
            Console.WriteLine($"Resultado da multiplicação: {resultadoMult}");

            int resultadoDiv = Matematica.Divisao(20, 4);
            Console.WriteLine($"Resultado da divisão: {resultadoDiv}");
        }
        catch (DivideByZeroException e)
        {
            Console.WriteLine(e.Message);
        }
    }
}
