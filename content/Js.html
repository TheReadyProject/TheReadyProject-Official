1. JavaScript

Módulo 1: Fundamentos de JavaScript

Aula 1: Introdução e Configuração do Ambiente

Objetivo: Introduzir o JavaScript e configurar o ambiente de desenvolvimento.

Conteúdo:

O que é JavaScript e suas aplicações:

JavaScript é uma linguagem de programação leve, interpretada e orientada a objetos com funções de primeira classe. É mais conhecida como a linguagem de script para páginas web, mas também é usada em muitas ambientes não navegadores, como Node.js.

Instalando Node.js para desenvolvimento em JavaScript:

  1. Acesse o site oficial do Node.js: nodejs.org.
  2. Baixe e instale a versão apropriada para o seu sistema operacional.

Configuração do ambiente de desenvolvimento:

Após a instalação, abra um terminal e verifique a instalação com os comandos node -v e npm -v.

Primeiro programa: "Hello, World!":

console.log("Hello, World!");

Aula 2: Variáveis e Tipos de Dados

Objetivo: Compreender variáveis e tipos de dados em JavaScript.

Conteúdo:

Variáveis: definição e convenções de nomenclatura:

let nome = "Alice";
let idade = 30;

Convenções de nomenclatura:

  • Use nomes descritivos.
  • Utilize camelCase para nomes de variáveis e funções.

Tipos de dados básicos: inteiros, floats, strings, booleanos:

let num = 10;  // Inteiro
let pi = 3.14;  // Float
let saudacao = "Olá, Mundo!";  // String
let isStudent = true;  // Booleano

Operações básicas com variáveis:

let soma = 10 + 5;  // Aritmética
let maior = 10 > 5;  // Comparação

Função `typeof` e conversão de tipos:

console.log(typeof nome);  // Saída: string
let numStr = "123";
let numInt = parseInt(numStr);

Aula 3: Estruturas de Controle (if, else, else if)

Objetivo: Aprender a utilizar estruturas de controle de fluxo.

Conteúdo:

Estruturas condicionais: `if`, `else`, `else if`:

if (idade >= 18) {
    console.log("Você é maior de idade.");
} else if (idade > 12) {
    console.log("Você é adolescente.");
} else {
    console.log("Você é criança.");
}

Exemplos práticos:

let numero = 7;
if (numero % 2 === 0) {
    console.log("O número é par.");
} else {
    console.log("O número é ímpar.");
}

Operadores de comparação e lógicos:

let idade = 20;
if (idade >= 18 && idade < 65) {
    console.log("Você é um adulto.");
}

Atividade:

Criação de um script simples que solicita ao usuário um número e verifica se ele é positivo, negativo ou zero.

let numero = prompt("Digite um número: ");
numero = parseFloat(numero);
if (numero > 0) {
    console.log("O número é positivo.");
} else if (numero < 0) {
    console.log("O número é negativo.");
} else {
    console.log("O número é zero.");
}

Módulo 2: Estruturas de Dados

Aula 1: Arrays, Listas e Objetos

Objetivo: Compreender e utilizar arrays, listas e objetos.

Conteúdo:

Arrays:

let frutas = ["maçã", "banana", "laranja"];
console.log(frutas[0]);  // Saída: maçã

Operações com arrays:

frutas[2] = "morango";  // Atualiza o valor no índice 2

Objetos:

let dadosAluno = {
    nome: "João",
    idade: 25,
    curso: "Engenharia"
};
console.log(dadosAluno.idade);  // Saída: 25

Operações com objetos:

dadosAluno.idade = 26;  // Atualiza o valor de uma propriedade
delete dadosAluno.curso;  // Remove uma propriedade

Atividade:

Criação de um programa que gerencia uma lista de tarefas, permitindo adicionar, remover e visualizar itens.

let tarefas = [];

while (true) {
    console.log("\nLista de Tarefas:");
    for (let tarefa of tarefas) {
        console.log(`- ${tarefa}`);
    }

    console.log("\nOpções:");
    console.log("1. Adicionar tarefa");
    console.log("2. Remover tarefa");
    console.log("3. Sair");

    let opcao = prompt("Escolha uma opção: ");

    if (opcao === "1") {
        let tarefa = prompt("Digite a tarefa: ");
        tarefas.push(tarefa);
    } else if (opcao === "2") {
        let tarefa = prompt("Digite a tarefa a remover: ");
        let index = tarefas.indexOf(tarefa);
        if (index !== -1) {
            tarefas.splice(index, 1);
        } else {
            console.log("Tarefa não encontrada.");
        }
    } else if (opcao === "3") {
        break;
    } else {
        console.log("Opção inválida.");
    }
}

Módulo 3: Funções e Módulos

Aula 1: Definição de Funções

Objetivo: Aprender a criar e utilizar funções em JavaScript.

Conteúdo:

Definição e chamada de funções:

function saudacao(nome) {
    console.log(`Olá, ${nome}!`);
}

saudacao("Alice");  // Saída: Olá, Alice!

Argumentos e parâmetros:

function soma(a, b) {
    return a + b;
}

let resultado = soma(3, 5);
console.log(resultado);  // Saída: 8

Parâmetros padrão e argumentos nomeados:

function saudacao(nome = "Visitante") {
    console.log(`Olá, ${nome}!`);
}

saudacao();  // Saída: Olá, Visitante!
saudacao("Alice");  // Saída: Olá, Alice!

Aula 2: Módulos e Pacotes

Objetivo: Aprender a organizar código em módulos e pacotes.

Importação de módulos:

// Arquivo: calculos.js
export function soma(a, b) {
    return a + b;
}

// Arquivo: main.js
import { soma } from './calculos.js';

let resultado = soma(3, 5);
console.log(`Resultado da soma: ${resultado}`);  // Saída: Resultado da soma: 8

Criação e uso de módulos:

// Arquivo: utilitarios.js
export function multiplicacao(a, b) {
    return a * b;
}

export function divisao(a, b) {
    if (b === 0) {
        throw new Error("O divisor não pode ser zero.");
    }
    return a / b;
}

// Arquivo: main.js
import { multiplicacao, divisao } from './utilitarios.js';

try {
    let resultadoMult = multiplicacao(4, 5);
    console.log(`Resultado da multiplicação: ${resultadoMult}`);

    let resultadoDiv = divisao(20, 4);
    console.log(`Resultado da divisão: ${resultadoDiv}`);
} catch (e) {
    console.log(e.message);
}
